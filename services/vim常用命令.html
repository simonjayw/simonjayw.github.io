# vim 常用命令

  • i 进入编辑

  • esc 推出编辑

  • / 搜索

    /foo\c \c 表示大小写不敏感 、\C 大小写敏感

  • u 撤销

  • ctrl + r 恢复上一次撤销

# 光标移动

  • 页面第一行 gg
  • 页面最后一行 shift + g
  • 行首 home （Mac 上是 FN + 左箭头）
  • 行末 end （Mac 上是 FN + 右箭头）
